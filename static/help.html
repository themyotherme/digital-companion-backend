Digital Companion Help Guide

Understanding Chat Modes

The Digital Companion offers three distinct modes for interacting with your documents and the AI. Each mode is designed for a specific purpose.

Mode AI Used? Internet Required? Knowledge Source Best For
Local No No Selected Document(s) Only (Basic Keyword Search) Quickly finding specific text in a document offline.
Smart Yes (GPT-3.5) Yes Selected Document(s) Only (AI Analysis) Getting intelligent summaries, answers, and analysis based strictly on your document's content.
SmartPlus Yes (GPT-4) Yes Document(s) + AI's General Knowledge Comprehensive research, creative tasks, and getting the best possible answer by combining your document with external information.

Managing Your Knowledge Base

The Knowledge Base is where you upload and manage the documents you want to chat with.

How to Use:

  • 1. Open the Manager: Click the Database icon in the top-right header to open the "Manage Knowledge Bases" dialog.
  • 2. Add Files: Click "Add File" to upload your documents. Currently, .pdf, .txt, and .json files are supported.
  • 3. Select Files: Check the box next to one or more files to select them for your chat session. The selected files will appear in the header.
  • 4. Chat: Choose your desired mode (Local, Smart, or SmartPlus) and start asking questions!

We're constantly working to improve the Digital Companion. Here are some features on the horizon:

  • Quiz Generation: Automatically create interactive quizzes from any document in your Knowledge Base.
  • Wider File Support: Adding support for more file types like Word, PowerPoint, and Excel.
